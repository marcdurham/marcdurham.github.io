The Problem:  

Running the VMware virtual machine causes the screen to flicker, or twitch, every 10-15 seconds, it's very irritating, I would describe it as "jolting and maddening", happening way too frequently to ignore.

The System:

I'm running VMware Fusion 8 on OS X Yosemite 10.10.5, my guest operating system is Windows 8.1, soon to be Windows 10 (It's upgrading as I'm typing this).   I'm running it on a MacBook Pro (Mid 2015) with a 15inch Retina display with 16GB RAM and AMD Radeon R9 M370X 2048 MB video.

The Solution:


  1. From the VMware menu, click Virtual Machine
  2. Click Settings
  3. Click Display
  4. Make sure Use full resolution for Retina display is checked.
  5. Make sure Scaled high resolution: is set to All View Modes
  6. Make sure Automatically adjust user interface size in the virtual machine is checked.


I actually had the same problem in VMware Fusion 6, and VMWare Fusion 7, I tried all three versions.
